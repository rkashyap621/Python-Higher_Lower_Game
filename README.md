# Python-Higher_Lower_Game
This project was inspired by the famous [higher lower game](https://www.higherlowergame.com). I used to play this game a lot and as I was starting to learn python recently, I thought creating a python version of the same. And here we are!

Later, I started to do some digging about the creator of the game and I found out that the [higher lower game](https://www.higherlowergame.com) was created by [Charles Burdett](https://dribbble.com/chxrles). He has done quite other cool projects, feel free to explore his [profile](https://dribbble.com/chxrles) in [dribbble](https://dribbble.com/).
